## Overview

A powerful Discord bot designed for server management, featuring moderation tools, custom commands, and entertainment features.

## Key Features

- **Moderation Suite**: Auto-mod, ban, kick, mute, and warn commands
- **Custom Commands**: Server admins can create custom responses
- **Music Player**: High-quality audio streaming from YouTube
- **Leveling System**: XP and role rewards for active members
- **Logging**: Comprehensive audit logs for server events
